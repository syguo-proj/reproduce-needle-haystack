# Posterior Concentration for Possibly Sparse Sequences

This repository reproduces the main posterior concentration results in Needles and Straw in a Haystack: Posterior Concentration for Possibly Sparse Sequences(Castillo & van der Vaart, 2012).

## Contents
- Reproduction_NSHPC_SparseSeq.Rmd: R code for computing posterior functionals as sparse signal estimators.
- Simulation-Report.pdf: Review of hierarchical model specification and main theoretical results, and simulation studies validating the posterior concentration behavior.
